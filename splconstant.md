# 💎 Phase 4C: Special Constants & Theoretical Notations (The System Rules)

These notations represent fundamental truths, universal rules, or special numbers that govern the entire mathematical system. They are the **laws of physics** of the mathematical world.

| Symbol | Name | Definition / Value | Role in Math |
| :---: | :--- | :--- | :---: |
| $\pi$ | **Pi** | The ratio of a circle's circumference to its diameter. ($\approx 3.14159...$) | Governs circular motion, geometry, and waves. |
| $e$ | **Euler's Number** | The base of the natural logarithm. ($\approx 2.71828...$) | Governs natural growth, decay, and continuous change (e.g., compound interest). |
| $i$ | **Imaginary Unit** | Defined as $\sqrt{-1}$. | Allows description of dimensions beyond the real number line (Complex plane). |
| $\aleph_0$ | **Aleph-Null** | The smallest infinite cardinal number. | Represents the size/count of the set of Natural Numbers ($\mathbb{N}$). |
| $\aleph_1$ | **Aleph-One** | The next cardinal number (assuming the Continuum Hypothesis). | Represents the size/count of the set of Real Numbers ($\mathbb{R}$). |
| $\implies$ | **Implication** | Logical "Implies" ($\text{If P, then Q}$). | Used in logic and proofs to show that one statement guarantees another. |
| $\iff$ | **If and Only If (IFF)** | Logical Equivalence. | Shows two statements are interchangeable (If P, then Q, AND if Q, then P). |
| $\forall$ | **Universal Quantifier** | "For All." | Used to state that a rule applies to **every** item in a set. |
| $\exists$ | **Existential Quantifier** | "There Exists." | Used to state that a rule applies to **at least one** item in a set. |
