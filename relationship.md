# 🎯 Phase 3C: Relational Symbols (The Conversion Tools)

These symbols define the relationship between two numbers (A and B). They act as the "measurement instruments" of your mathematical machine, telling you *how* the input energy was converted.

| Symbol | Name | Description | Example (Input A, B) |
| :---: | :--- | :--- | :---: |
| $=$ | **Equal To** | The values are exactly the same. | $5 = 5$ |
| $\neq$ | **Not Equal To** | The values are different. | $5 \neq 7$ |
| $<$ | **Less Than** | The value on the left is smaller than the value on the right. | $5 < 7$ |
| $>$ | **Greater Than** | The value on the left is larger than the value on the right. | $10 > 8$ |
| $\le$ | **Less Than or Equal To** | The left side is smaller or the same. | $7 \le 7$ |
| $\ge$ | **Greater Than or Equal To** | The left side is larger or the same. | $10 \ge 8$ |
| $\approx$ | **Approximately Equal To** | The values are very close in value (often used for irrational numbers). | $\pi \approx 3.14$ |
| $\propto$ | **Proportional To** | One value changes directly in relation to the other. | $y \propto x$ (Meaning $y = kx$ for some constant $k$). |
