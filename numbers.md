# 📄 Phase 1: The Number Sets (The Domain of Energy)

This table defines *where* your input number (0-9) lives. It determines the mathematical set that contains the number.

| Symbol | Name | Definition / Range | Description | Example (Input 7) |
| :---: | :--- | :--- | :--- | :---: |
| $\mathbb{N}$ | **Natural Numbers** | $\{1, 2, 3, \dots, \infty\}$ | The pure counting numbers. | $7 \in \mathbb{N}$ |
| $\mathbb{W}$ | **Whole Numbers** | $\{0, 1, 2, 3, \dots, \infty\}$ | Natural numbers plus zero. | $7 \in \mathbb{W}$ |
| $\mathbb{Z}$ | **Integers** | $\{\dots, -2, -1, 0, 1, 2, \dots\}$ | All whole numbers and their negative counterparts. | $7 \in \mathbb{Z}$ |
| $\mathbb{Q}$ | **Rational Numbers** | $\{p/q \mid p, q \in \mathbb{Z}, q \neq 0\}$ | Can be written as a simple fraction. Decimals terminate or repeat. | $7 = 7/1$. $7 \in \mathbb{Q}$ |
| $\mathbb{I}$ | **Irrational Numbers** | Non-repeating, infinite decimals. | Cannot be written as a simple fraction. (e.g., $\pi$, $\sqrt{2}$) | $7$ is NOT irrational, but $7+\pi$ IS. |
| $\mathbb{R}$ | **Real Numbers** | $\mathbb{Q} \cup \mathbb{I}$ (All $\mathbb{Q}$ and $\mathbb{I}$) | The entire continuous number line. | $7 \in \mathbb{R}$ |
| $\mathbb{C}$ | **Complex Numbers** | $\{a + bi \mid a, b \in \mathbb{R}\}$ | Real part ($a$) and Imaginary part ($b$). $i = \sqrt{-1}$. | $7 = 7 + 0i$. $7 \in \mathbb{C}$ |
